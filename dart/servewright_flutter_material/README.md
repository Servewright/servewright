# servewright_flutter_material

Reference **Material adapter** for Servewright Flutter. Registers standard V1 primitives with accessibility semantics.

## Install

```yaml
dependencies:
  servewright_flutter: ^0.1.0
  servewright_flutter_material: ^0.1.0
```

## Usage

```dart
registerMaterialPrimitives(registry);
```
