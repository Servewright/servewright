# servewright-core

Framework-agnostic JVM core for Servewright: semantic view model, validation, action routing, transitions, JSON serialization.

## Maven

```xml
<dependency>
  <groupId>io.servewright</groupId>
  <artifactId>servewright-core</artifactId>
  <version>0.1.0-SNAPSHOT</version>
</dependency>
```

No Spring dependency. Use `servewright-spring-boot-starter` for HTTP/SSE integration.
